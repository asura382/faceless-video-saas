API ENDPOINTS:
--------------
GET    /health                 - Health check
POST   /api/videos             - Create new video
GET    /api/videos             - List all videos
GET    /api/videos/{id}        - Get video details
DELETE /api/videos/{id}        - Delete video
GET    /api/videos/{id}/download - Download MP4
GET    /media/{filename}       - Serve media files

FREE SERVICES USED:
-------------------
1. HuggingFace Inference API - Script generation (~1000 req/day free)
2. gTTS - Text-to-speech (completely free)
3. Pexels API - Stock videos (200 req/hour free)
4. Pixabay API - Stock videos (100 req/min free, fallback)
5. Render - Hosting (750 hrs/month free)
6. Vercel - Frontend hosting (100GB bandwidth free)
7. Render PostgreSQL - Database (1GB free)

VIDEO PIPELINE:
---------------
1. User enters topic
2. AI generates script (HuggingFace)
3. Script converted to audio (gTTS)
4. Stock videos fetched (Pexels/Pixabay)
5. Subtitles generated
6. FFmpeg renders final video (9:16 format)
7. User downloads MP4

DEPLOYMENT:
-----------
Backend: Render (Docker + PostgreSQL)
Frontend: Vercel (Next.js static export)

See DEPLOYMENT.md for detailed instructions.
